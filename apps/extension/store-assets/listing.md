# Chrome Web Store 上架资料

## 扩展名称
Chord · 回响

## 简短描述（≤132字符）
每日唤醒一条收藏，三向决策帮你识别真实兴趣。不是书签管理器，是内容与自我的对话工具。

## 详细描述

你收藏了多少篇「以后要读」的文章？那个书签夹里，是你的真实兴趣，还是一个个未兑现的承诺？

**回响**每天从你的书房里唤醒一条内容，轻声问一句：这个，你还记得吗？

然后，你只需要做三个决定中的一个：

📌 **留下来** — 它还有价值，保留，但不急着处理
✓ **用过了** — 它真的派上用场了，记录怎么用的
🌸 **放手** — 承认这是幻觉兴趣，轻盈地放手

---

### 核心功能

**书房（候响室）**
- 保存任意网页，自动识别内容型与工具型链接
- 智能分类：文章归书房，Gmail / Figma 等工具归快速入口
- 批量导入现有书签，30 秒清点数字积压

**每日回响**
- 加权随机算法，优先唤醒积压最久、从未处理的内容
- AI 生成个人化「回响问句」，激活真实的思考
- 三向决策 + 精美动效（樱花飘落 / pin 压印 / 涟漪对勾）
- 连续天数 streak，培养整理习惯

**兴趣地形**
- 自动聚类书房内容为 5-8 个主题
- 气泡大小 = 收藏量；虚线边框 = 幻觉兴趣，实线 = 真实热情
- 可视化你真正在意什么

**隐性自我**
- 基于行为数据生成 4 条洞察发现
- 真实热情率、本月放手数、处理完成度

---

### 隐私承诺

- 「只存本地」模式：零网络请求，可用 DevTools 验证
- 私人注释内容永远不离开你的设备
- 支持数据导出（JSON）和一键清空
- AI 模式仅传文章标题和域名，API Key 本地加密存储

---

### AI 支持（可选）

支持接入 Claude / OpenAI / Gemini / 豆包 / Kimi，只需填入自己的 API Key。
不填 Key 也能完整使用，TF-IDF 离线算法处理聚类和问句生成。
